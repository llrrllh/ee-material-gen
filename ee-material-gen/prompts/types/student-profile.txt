## 素材类型：学员画像（student-profile）

### 规格与技术要求
- 尺寸：A4比例（210mm × 297mm），共2页
- 实现：HTML中用两个 class="a4-page" 的容器，每个容器设置 width:794px; height:1123px
- 格式：完整独立HTML，CSS内联，适合打印/另存为PDF
- 打印优化：加入 @media print { } 样式，确保分页正确

### 第1页内容：数据可视化画像
1. **页眉**
   - 左上：课程名称
   - 右上：复旦大学管理学院 | FDSM
   - 横线分隔

2. **标题区**
   - 大标题："学员画像"或"同学圈层一览"
   - 副标题：当前期数 + 年份

3. **行业分布（环形图/饼图）**
   - 使用SVG绘制，不依赖外部库
   - 显示5-7个行业分类（制造业/金融/医疗/科技/消费等）
   - 每个扇形配色使用课程色系
   - 旁边列出图例：行业名 + 百分比

4. **职位级别分布（横向条形图）**
   - SVG或纯CSS实现
   - 显示：董事长/CEO、总裁/总经理、VP/副总裁、创始人、其他
   - 每条用课程主色系，宽度对应比例

5. **地域分布（可选）**
   - 简化地图或省份列表 + 数量
   - CSS网格或flex布局

6. **一组核心数字**
   - 4个大数字卡片：平均年龄、平均企业规模/营收区间、来自X个城市、X个行业
   - 大字号辅色数字 + 小字描述

### 第2页内容：典型画像描述 + 同频感染
1. **典型学员故事（3个人物画像）**
   - 每个：圆形头像占位（CSS渐变，标注职位首字）+ 职位/行业 + 2-3行学员感言或画像描述
   - 引发"这说的就是我"的共鸣感

2. **学员企业样本展示**
   - 展示10-15个代表性企业/行业标签
   - 使用标签云（CSS flex换行）或带边框的胶囊标签

3. **同学网络价值区块**
   - 标题："这个课堂，是一个真实的生态"
   - 3个价值点：资源对接/产业合作/长期圈层
   - 图标+文字，简洁排列

4. **页脚**
   - 申请咨询方式
   - 版权：复旦大学管理学院 EE © 2025

### 视觉风格规范
- A4白底，专业文件感，适合打印
- 数据图表用课程品牌色系（主色+辅色渐变）
- 整体干净、数据密度高但不拥挤
- 标题用课程主色，数字用辅色（金色/强调色）高亮
- 图表必须是"无外部依赖"的SVG或CSS实现
