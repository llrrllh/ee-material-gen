# MCN创新发展"星"计划

## 课程定位
聚焦MCN机构与新媒体产业，结合复旦管院营销学科与行业头部实践，帮助MCN生态从业者掌握可持续发展的商业模式与运营增长方法论。

## 目标学员
MCN机构/品牌企业/电商企业高管、传统媒体从业者、新媒体创业者、MCN投资人；覆盖内容产业链上下游的核心决策者和管理者。

- MCN行业趋势与商业模式重构
