# 复旦大学生命健康产业领军/领航计划

> **注**：课程在不同材料中出现两个名称——"领军计划"（课程画册）和"领航计划"（汇总表）。根据学费（30.8万）和学制（约12个月）一致，判断为同一课程，暂以"领军计划"为主名称，待确认后统一。

## 课程定位
推进产、学、研、医、资全链路深度融合，以前沿科研成果、临床创新、管理创新与资本赋能，全方位协同助力生命健康领域创新成果落地转化。

## 目标学员
医疗科技/医疗器械/生物制药/医疗投资机构高管，学员平均年龄44岁；覆盖生命健康头部企业、医疗机构、科研机构管理者，以及企投家、产业投资机构、风险投资机构合伙人。

## 核心师资
- 高解春（复旦医院管理研究所所长）
- 陆雄文（复旦大学管理学院院长）
- 漆远（复旦人工智能创新与产业研究院院长）
- 张文宏（复旦大学华山医院感染科主任）
- 邬惊雷（上海市卫生健康委原主任）
- 联合：复旦医院管理研究所 + 生命科学学院

## 学制与学费
- 学制：约12个月，每月2-3天共11次课，含国内+海外移动课堂
- 开班时间：2025年9月（参考数据）
- 学费：¥308,000
- 上课地点：复旦大学管理学院（上海）+ 国内及海外移动课堂

## 课程模块/大纲
围绕"创新链 × 产业链 × 资金链"三链融合：
- 前沿科研成果与临床创新
- 管理创新与运营提效
- 资本赋能：生命健康领域的投资逻辑与融资路径
- 复旦优质医疗产业资源整合与对接
- 产、学、研、医、资全链路深度联结

## 招生话术要点
- 核心卖点：产学研医资"五位一体"，是市场上少有能真正打通全链路资源的生命健康高管课程
- 张文宏、高解春等业内顶级学者加持，医疗从业者认知度高，话术中可直接点名
- 学员平均年龄44岁，客群成熟，沟通风格应更注重深度和专业性
- 参考话术：*"生命健康行业最难的不是技术，而是如何把科研成果跨越'死亡谷'转化为产业。这个课程的价值在于，它同时把医院院长、科研院所负责人、投资机构合伙人放在同一个教室里——这是真正意义上的资源打通，而不是课程噱头。"*

## 关键运营信息
- 内部简称"大健康"
- 学费30.8万，目标客群为生命健康领域高净值管理者
- 海外移动课堂是额外吸引力，具体目的地待确认
