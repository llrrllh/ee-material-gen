# 复旦医疗科技企业领创营

## 课程定位
专为医疗科技创业企业核心团队设计，以"临床-管理-资本"三维整合，帮助天使轮至B轮阶段的医疗科技创业者跨越死亡谷，实现快速成长。

## 目标学员
天使轮至B轮阶段的医疗科技创业企业创始人及核心团队；包括医疗器械、医疗科技、生物制药方向的创业者。

## 招生话术要点
- 聚焦早期医疗科技创业企业，是细分赛道专属产品，差异化极强
- 强生、美敦力前高管是重要背书，医疗器械创业者圈子中认知度高
- 高解春所长的临床资源网络是核心价值
- 参考话术：*"医疗科技创业最难跨越的是从'技术可行'到'商业落地'的鸿沟。这个营不教通用管理，而是专门针对天使到B轮阶段的医疗科技团队，帮你打通临床、商业和资本三条线——这是复旦医院管理研究所+强生美敦力前高管团队才能给到的组合。"*
