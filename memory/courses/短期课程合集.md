# 短期课程合集（启智学堂）

## 课程定位
以3-5天短期集中式培训为主要形式，聚焦垂直行业或专项管理议题，降低时间成本，实现精准赋能；是复旦EE面向各行业高层管理者的轻量化入口产品。

## 目标学员
各行业高层管理者；尤其适合：时间紧张但有阶段性学习需求的高管、希望了解复旦EE体系后再决定长期项目报名的潜在学员、企业定向送培的特定议题培训需求。

## 核心师资
按各短期课程单独配置，通常由对应领域的复旦管院教授主导，搭配业界实践导师。

## 学制与学费
- 学制：3-5天短期集中式
- 学费：各课程单独定价（差异较大，从数千元至数万元不等）
- 上课地点：复旦大学管理学院（上海）

## 课程模块/大纲
启智学堂旗下短期课程涵盖多个主题，已知包括：
- **企业风险管理课程**（3天，¥16,800，详见`企业风险管理.md`）
- **全球数字经济与加密资产战略课程**（2天，¥12,800，详见`全球数字经济与加密资产.md`）
- **非财务高管的财务管理课程**（6天，¥23,900，详见`非财.md`）
- 其他专项主题课程（以最新官网和招生简章为准）

## 招生话术要点
- 短期课程是复旦EE的"轻量入口"，可作为与客户建立初步信任的低风险选择
- 话术策略：引导有长期学习意愿的客户先参加短期课程体验，后续转化至长期项目
- 参考话术：*"如果您还没确定是否要参加完整的年制课程，可以先考虑我们的启智学堂短课——3-5天，聚焦一个具体议题，成本和时间投入都低，但能直接感受复旦管院的教学质量和同学圈层。"*

## 关键运营信息
- 短期课程招生频次高、周期短，需要持续跟进最新开班计划
- 启智学堂品牌可与长期项目形成联动，建立课程体系的层次感
- 适合作为企业BD的"测试球"产品，企业先送少量员工参加短课，满意后再批量合作
