# 企业定制内训服务

## 课程定位
面向企业（B端），依据企业战略需求和人才发展目标，量身定制系统性管理培训方案；复旦管院作为外部智囊团，提供从需求分析到课程设计到落地实施的全链路服务。

## 目标学员
企业（B端）整体，无固定招生对象；典型采购决策人为：
- 世界500强/大型外企亚太区总裁
- 处于高速扩张期的拟上市企业实控人
- 面临产业升级的传统制造业龙头管理层
- 企业CHRO/CHO/人才发展负责人

## 核心师资
复旦大学管理学院200+学术师资 + 100+业界实践导师；根据企业需求精准匹配，典型合作案例包括博世（Bosch）深度内训项目。

## 学制与学费
- 学制：按企业需求定制，典型方案3-6天；也可设计为多期连续项目（如每月1次，持续6-12个月）
- 学费：¥18-20万/天（含需求分析、课程设计、师资、教材等全包价）
- 授课地点：可赴企业现场，或在复旦管院校区

## 课程模块/大纲
按需定制，常见模块方向：
- 战略管理与高管领导力
- 组织转型与变革管理
- 财务思维与资本决策
- 数字化转型与AI战略
- 跨界对标与博士团入企诊断（创新模块）
- 沙盘模拟与战略共识工作坊

## 招生话术要点
- **【大型外企亚太区总裁】本土化战略落地**：
  > "对于大型跨国企业而言，全球战略的本土化落地离不开一支懂中国市场、具备跨文化领导力的高管团队。参考我们深度运营的'博世内训项目'，复旦管院能够深度调研您的业务痛点，将经典的经管理论与您的企业战略深度融合，量身定制专属的内训体系。"
- **【拟上市企业实控人】高管团队能力压实**：
  > "企业在冲刺IPO阶段，单兵作战能力必须转化为组织能力。通过高管团队的集中内训，我们不仅能快速补齐他们在财务风控、合规治理和组织管理上的短板，更能在封闭式的学习研讨中，帮助您的核心班子达成高度一致的'战略共识'，大幅降低内部沟通摩擦。"
- **【传统制造业龙头】破局第二曲线**：
  > "转型期企业面临的最大阻力往往来自内部认知的固化。复旦管院的内训项目，将引入'跨界对标'和'博士团入企诊断'等创新模块。我们不仅是培训机构，更是您的外部智囊团，通过引导式的工作坊，帮助您的团队共同碰撞出切合实际的转型破局方案。"

## 关键运营信息
- 定价18-20万/天，客单价极高，BD周期较长，需要高层对接和多轮沟通
- 博世内训是典型标杆案例，可在与大客户沟通时引用
- 企业定制内训的价值不只在课程本身，更在于"复旦智库支持"和"外部视角诊断"，话术需强调这一层
- 内训服务可作为与大企业建立长期合作关系的切入点，逐步扩大合作范围
