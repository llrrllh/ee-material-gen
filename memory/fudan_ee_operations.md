# Memory: 复旦EE日常工作核心

## 一、工作模式 Profile (Lila)
- **核心职能**: 围绕高净值人群，打通“前端招生营销”到“后端精细化运营”的完整闭环。
- **核心方法**: 盘人（客户分析）、盘资源（内外部）、盘逻辑（SOP与流程优化）。
- **文档标准**: 美观且有内容，逻辑清晰，结构化，拒绝空话和“大词”。

## 二、核心业务模块

### 1. 招生（销售）与营销
- **SOP**:
  - **客户分析**: 深入分析简历，识别“水分”与“干货”，判断真实诉求与痛点。
  - **价值匹配**: 将客户痛点与课程价值（尤其是圈层、资源价值）做精准匹配。
  - **逼单策略**: 采用“先捧后压，顺水推舟”的策略，先认可价值，再点出焦虑，最后给出解决方案（课程）。
- **关键物料**: 课程画册（命名规范：【课程手册】课程全名【版本号】）。

### 3. 文案撰写
- **适用场景**: 开学发言稿、学员通知、班级群公告、招生推文、课程介绍等。
- **要点**:
  - 文案量大，贯穿招生和运营全流程，需提前规划写作节点，不要临时抱佛脚。
  - 风格要求：精准有力、得体专业，避免空话套话。根据受众（师资、学员、潜客）调整语气。
  - 开学前重点文案清单：领导发言稿、开幕词、学员手册介绍页、班级群欢迎词、进校通知。
  - 可交给助手起草初稿，璐桁审核定稿。

### 2. 项目运营
#### A. “开学典礼”模式 (适用于新班开学)
- **核心目标**: 建立身份认同，完成破冰，启动项目。
- **关键动作**: 
  - 颗粒度极细的Checklist，分“启动规划 → 信息设计 → 制作采购 → 最终筹备”四阶段。
  - 包含教学、市场、学员、典礼晚宴四大块。
  - **模板文件**: `AI领创2期_运营Checklist.xlsx` (可作为未来所有新班开学的母版)。

#### B. “日常授课”模式 (适用于企业家班等常规模块)
- **核心目标**: 保证课程顺利交付，提供高品质的学员体验。
- **关键动作**: 
  - 围绕“场地、供应商、教务、通知、师资”五大板块展开。
  - 重点在于课前的细节检查与多方沟通。
  - **模板文件**: `本周末上课准备Checklist_46&47期.md` (将基于此文件生成标准化SOP)。
- **自动化**: 可根据上课日期，一键生成所有课前准备的提醒事项。
